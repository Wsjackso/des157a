Post 2 - Post 2 - Overlays Design Pattern Research

Jaunary 30,2025 Best Practices for Modals / Overlays / Dialog Windows

The article “Best Practices for Modals/Overlays/ Dialog Windows” covers how to make overlays more effective and user-friendly designs. One thing I learned is how important it is to give users clear ways to exit the overlay like a close button, clicking outside the overlay, or pressing the Escape key. If people feel trapped in a overlay, it’s a bad user experience. Another interesting point was about focus. Overlays should dim the background and keep the focus on the modal window to avoid distractions, especially for accessibility. I also liked the idea of using actionable buttons with clear labels, like “Confirm” or “Cancel,” instead of something like “OK.”Overall, I think this article had good tips and it helped me understand how good overlay design keeps the user in control while interacting with web designs.

Post 1 - Best Practices and Ideas

Jaunary 25,2025 Best practies for form design Hungry Root

The article “Best Practices for Form Design” provides an overview of how to create more effective, user-friendly forms. Going through the 20 practices gave me a much better understanding of what makes good form design. As a designer, every decision matters when creating an experience for others. For example, using principles like “multiple steps” helps reduce the user’s cognitive load by breaking up the process and showing how many steps are left. This approach allows users to focus on the current step without feeling overwhelmed.

Another point I found interesting was about placeholder text. It can hurt usability because once users start typing, they might forget what was written and have to delete their input to see it again. To avoid this, adding a description below the input field is a better user friendly design choice. Overall, this article was a cool guide for improving form design.

The website I chose with good user-friendly form design is Hungryroot. Hungryroot offers curated meals that you can order based on personalized recommendations from a quiz. The quiz is engaging and only takes a 15 minutes to complete. By answering a few questions, you get a curated taste profile, which makes the experience feel personalized and fun.
